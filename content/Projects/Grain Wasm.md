* Source: https://github.com/spotandjake/Grain-Wasm

Grain Wasm is a simple and fast Wasm parser and builder written in grain-lang. This library was mostly written in a single day, it allows for both parsing and emitting Wasm bytecode and implements about 99% of the MVP spec. I have future plans to rewrite this library using a more modular manner for [[Projects/Brisk|Brisk]]
