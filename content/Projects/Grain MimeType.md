## Description
* Source: https://github.com/spotandjake/Grain-mimeType

This is a very simple module for looking up mimeTypes based on file extensions it is generated using a simple grain script that reads a downloaded copy of `db.json` from  [Mime-DB](https://github.com/jshttp/mime-db)
