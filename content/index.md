### What is this
This is my public knowledge base, where I share research, technical insights, and notes on my learning adventures.

### How it works
I write and edit all of my notes in markdown through Obsidian, as I am not able to make all of my notes public these are uploaded to a private GitHub repo. Every commit a GitHub action runs that scans my notes for publishable files and copies them over to my quartz directory for hosting.
