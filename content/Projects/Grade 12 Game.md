## Description
* Source: https://github.com/spotandjake/Grade12Game

This was a very primitive tower defense game I built using XNA in cSharp for my grade 12 game project. I implemented a custom procedural terrain generator along with an extensible tower and enemy system allowing for endless hours of gameplay, unfortunately I didn't have much time for this project so I couldn't do a lot in terms of assets or terrain (I was looking into marching cubes).
